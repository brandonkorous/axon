## [0.1.0] - 2025-03-21

### Added
- Multi-agent orchestration with persistent personas
- Voice-first interface with per-agent voice synthesis
- Real-time Huddle (boardroom) sessions with multiple advisors
- Neural memory system with Obsidian-compatible vaults
- Multi-LLM support: Anthropic Claude, OpenAI, Ollama (local)
- Multi-org support with isolated agent configurations
- Task runner for autonomous agent code execution
- Central dashboard with agent status, decisions, and inbox
- Achievement system for agent and user milestones
- Audit logging for all agent actions
- YAML-based persona configuration
- Docker Compose deployment
- SQLite (default) and PostgreSQL database support
