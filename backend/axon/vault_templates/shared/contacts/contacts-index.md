# Contacts

Organization contacts. Each contact file should include frontmatter with:
- `name` — full name
- `email` — email address
- `discord_id` — Discord user ID (optional)
- `phone` — phone number (optional)
- `role` — relationship or role (e.g., investor, partner, customer)
- `company` — company name (optional)
