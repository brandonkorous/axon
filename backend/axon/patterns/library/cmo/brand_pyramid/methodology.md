# Brand Pyramid

The Brand Pyramid is a layered framework for defining brand identity from the ground up. At the base are tangible attributes (what the product does). Above that sit functional benefits (why those attributes matter). Then emotional benefits (how it makes people feel). Then values (what the brand believes). At the apex is the brand personality or essence — the single idea that ties everything together.

Apply this when building a new brand, refreshing an existing one, or when team members describe the brand inconsistently. The pyramid creates a shared reference that keeps messaging, design, and product decisions coherent.

**Steps to apply:**
1. **Attributes** — List the factual, tangible features of the product or service.
2. **Functional benefits** — Translate each attribute into a concrete benefit for the user.
3. **Emotional benefits** — Identify how those functional benefits make the user feel.
4. **Values** — Define the beliefs and principles the brand stands for.
5. **Personality/Essence** — Distill everything into a single phrase or character description.

**Common mistakes:** Starting from the top (aspirational personality) without grounding it in real attributes. Listing features without translating them into benefits. Creating a pyramid that sounds good internally but does not resonate with actual customers.
