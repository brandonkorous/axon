# Hook Model

The Hook Model describes how products create habits through a four-phase loop: Trigger, Action, Variable Reward, and Investment. External triggers (notifications, emails) eventually become internal triggers (boredom, anxiety). The action must be simple. The reward must vary to maintain interest. The investment stores value that makes the next cycle more likely. Each pass through the loop strengthens the habit.

Apply this when designing onboarding flows, engagement features, or retention strategies. It is especially relevant for products that depend on repeat usage rather than one-time purchases.

**Steps to apply:**
1. **Trigger** — Identify the internal trigger (emotion/situation) your product addresses. Design external triggers to bridge until the habit forms.
2. **Action** — Make the intended behavior as easy as possible (fewer clicks, less thinking).
3. **Variable Reward** — Provide rewards that satisfy the trigger but vary enough to sustain curiosity.
4. **Investment** — Ask the user to put something in (data, content, preferences) that improves the next experience.

**Common mistakes:** Relying only on external triggers without building toward internal ones. Making the action too complex. Providing predictable rewards that lose novelty. Asking for investment before delivering value.
